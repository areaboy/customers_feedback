Customers Feedback AI
An interactive Web Applications for analyzing and visualizing Sentimental Analysis on 
Users/Customers Feedback Reports powered Azure Text-Analytics (Sentimental API Cognitive Services)


How to Run This Application

1.)Ensure that you have Xampp Server is Install.

2.)Ensure that Php and Mysql is running

3.)Edit data6rst.php and data6rst1.php to reflect database credentials respectively

4.) Import microsoftazureai_customersfeedback.sql database

5.) Edit both feedback_analyzed.php, feedback_reports.php, feedback_report2.php and add  your own Azure Text-Analytics Sentimental Opinon Mining API keys were appropriates respectively 


6.) Call up the browser and run the code at  http://localhost/feedback_AzureAI
